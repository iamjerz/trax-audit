# What's New — Version 1.0.0.24

A quick summary of the latest updates to Audit Ops.

### New
- **Home page** with quick links and key stats when you log in.
- **Triad Dashboard** showing pass/fail results and team performance.
- **Activity log** so admins can see logins and changes (Audit Trail).
- **Extension Details** page for admins to manage extension versions.

### Improved & Fixed
- QA Monitoring dashboard now shows the correct totals and scores.
- The show/hide password button now works on the login and password screens.
- You now get a clear message when a login fails.
- Admins can reset a user's password from the user page.

### For Your Account
- If you're still using the default password, you'll be asked to set your own at next login.
- You'll only see the sections your account has access to.

Questions? Contact your administrator.
